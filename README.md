Learning Basic VIM/TMUX

Source: https://docs.swift.org/swift-book/LanguageGuide/TheBasics.html

Tmux: https://github.com/anishathalye/dotfiles/blob/master/tmux.conf

- Vertical split: Bind + %
- Select Pane: Bind + VIM

- tmux kill-server
- vim ~/.tmux.conf
